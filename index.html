Menggambar Bentuk
  • Terdapat tombol Line, Square, Rectangle, dan Polygon yang dapat digunakan untuk menggambar bentuk
  • Tekan salah satu tombol tersebut, lalu tekan satu titik di kanvas untuk mulai menggambar
  • Untuk Line, diperlukan dua kali klik untuk menentukan titik awal dan titik akhir garis
  • Untuk Square, diperlukan dua kali klik untuk menentukan ukuran dari persegi
  • Untuk Rectangle, diperlukan dua kali klik untuk menentukan ukuran dari persegi panjang
  • Untuk Polygon, gunakan klik kiri untuk menggambar sudut-sudut polygon. Fitur polygon minimal membentuk segitiga sehingga diperlukan minimal tiga kali klik, Anda dapat mengakhiri pembuatan polygon dengan melakukan klik tengah.
Mengganti Attribut Warna
  • Terdapat Color Picker yang dapat digunakan untuk mengganti warna bentuk yang akan digambar
  • Untuk mengganti warna bentuk yang telah digambar, gunakan tombol Change Color lalu pilih warna yang diinginkan di Color Picker, kemudian tekan ke bentuk yang diinginkan untuk diganti warnanya
Memodifikasi Bentuk
  • Bentuk-bentuk yang sudah digambar dapat dimodifikasi bentuknya dengan menekan titik sudut yang ingin digeser, kemudian menekan lokasi baru untuk titik sudut tersebut dalam kanvas
  • Bentuk yang sudah digambar dapat dimodifikasi ukurannya dengan menuliskan skala dari 0 sampai 5 kemudian menekan resize dan menekan bentuk yang ingin diubah ukurannya.
Menyimpan dan Membuka Kembali Hasil Gambar
  • Untuk menyimpan gambar pada canvas, dapat dengan menekan tombol Download. File JSON akan tersimpan pada folder tujuan download browser Anda
  • Untuk membuka kembali gambar yang pernah disimpan, dapat dengan menekan tombol Choose File, kemudian memilih file JSON yang sebelumnya pernah disimpan
